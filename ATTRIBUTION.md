# Attribution and pinned artifacts

## OpenMemories

Camera lifecycle, key handling, media API references, and compile-time Sony signatures derive from ma1co's PMCADemo and OpenMemories Framework, commit family `f8df3504d8`.

```text
014fccb8313ea6ca8904e3b2be713dfe1b1e76e0860577b3b1665e57f4571901  openmemories-framework-f8df3504d8.jar
f0beeca857982feb7a39da01d548910d8c4202fb8c6fadd4060ac8784905a6e4  openmemories-stubs-f8df3504d8.jar
```

The stubs dependency is `provided` and must not shadow device implementations.

## PicoJPEG

PicoJPEG is Rich Geldreich's public-domain JPEG decoder, originally pinned from upstream commit `8ab33a909b4115ace4a952b7ffcb64c350f9298d`. The local copy includes reduced color-block sampling and a WaveSnap extension for luminance-only DC decoding.

```text
62c2999fa13b34906729d6a37a0493cd7e9a4c728f7eba83e7304fd35ffc8174  picojpeg.c
0d485a9a15469a7fb30c1b6774268675448607589de86b451dd63fb3a8491af7  picojpeg.h
```

## stb_image_write

`stb_image_write.h` v1.16 is pinned from nothings/stb commit `31c1ad37456438565541f4919958214b6e762fb4` and used under its MIT option.

```text
cbd5f0ad7a9cf4468affb36354a1d2338034f2c12473cf1a8e32053cb6914a05  stb_image_write.h
```

RetroLens compiles it with `STBI_WRITE_NO_STDIO` and writes through a fixed-capacity callback buffer.

## Visual assets

The native bitmap glyphs, icons, panels, and procedural thumbnails are original code-native RetroLens assets. No Sony branding or copyrighted game/film assets are used.
